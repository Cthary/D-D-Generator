## Project Overview
The D&D Character Generator is a project built to create dynamic Dungeons & Dragons characters with custom lore. It leverages OOP principles, follows [[SOLID]] design principles, and integrates with the [[ChatGPT]] API to generate custom lore for each character. 
### Technologies:
- **[[Python]]** for core logic and API. 
- **[[Flask]]** for the web API. 
- **[[ChatGPT]] API** for dynamic lore generation. 
- **[[Pytest]]** for testing. 
- **[[Docker]]** for deployment.
